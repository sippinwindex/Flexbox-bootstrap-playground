Getting Started with HTML

Beginner

HTML (HyperText Markup Language) is the foundation of web development. It provides structure and meaning to web content through elements and attributes.

What You'll Learn
  • HTML document structure and syntax
  • Essential elements for text, media, and forms
  • Interactive examples with live preview and source code
  • Copy-paste ready code snippets
Your First HTML Document

Hello, World!

Welcome to my first web page.

HTML 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My First Web Page</title>
</head>
<body>
    <h1>Hello, World!</h1>
    <p>Welcome to my first web page.</p>
</body>
</html>

Text Formatting

Text Formatting Elements

This is a paragraph with bold text and italic text.

You can also use highlighted text, small text, and inline code.

"This is a blockquote for longer quotations." - Author Name
HTML 
<p>This is a paragraph with <strong>bold text</strong> and <em>italic text</em>.</p>

<p>You can also use <mark>highlighted text</mark>, <small>small text</small>, and <code>inline code</code>.</p>

<blockquote>
    "This is a blockquote for longer quotations."
    <cite>- Author Name</cite>
</blockquote>
Best Practice

Use <strong> and <em> instead of <b> and <i> for semantic meaning, not just visual styling.

Links & Media

Beginner

Links

Different Types of Links

External Link (opens in new tab)

Internal Link (jump to section)

Email Link

Phone Link

HTML 

<a href="https://example.com" target="_blank" rel="noopener">
    External Link (opens in new tab)
</a>


<a href="#section-id">Internal Link (jump to section)</a>


<a href="mailto:contact@example.com">Email Link</a>


<a href="tel:+1234567890">Phone Link</a>
Important

Always include the alt attribute for accessibility. It describes the image for screen readers and shows when images fail to load.

Forms & Input

Intermediate
Contact Form with Various Input Types
HTML 
<form action="/submit" method="POST">
    <!-- Text Input -->
    <label for="name">Full Name:</label>
    <input type="text" id="name" name="name" required>

    <!-- Email Input -->
    <label for="email">Email Address:</label>
    <input type="email" id="email" name="email" required>

    <!-- Textarea -->
    <label for="message">Message:</label>
    <textarea id="message" name="message" rows="4"></textarea>

    <!-- Submit Button -->
    <button type="submit">Submit Form</button>
</form>
Form Best Practices
  • Always use <label> elements with form controls
  • Include required attribute for mandatory fields
  • Use appropriate input types (email, tel, date, etc.)
  • Group related inputs with <fieldset> and <legend>

Semantic HTML5

Intermediate

Semantic HTML uses elements that clearly describe their meaning and purpose, improving accessibility and SEO.

Semantic HTML5 Page Structure
Navigation: Home | About | Contact

Welcome to Our Website

Latest News

Published on January 15, 2024

Article content goes here...

Related Links

Additional resources and information.

© 2024 Company Name. All rights reserved.

HTML 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Semantic HTML Example</title>
</head>
<body>
    <header>
        <nav>
            <ul>
                <li><a href="#home">Home</a></li>
                <li><a href="#about">About</a></li>
                <li><a href="#contact">Contact</a></li>
            </ul>
        </nav>
    </header>

    <main>
        <section id="home">
            <h1>Welcome to Our Website</h1>
            <article>
                <h2>Latest News</h2>
                <p>Published on <time datetime="2024-01-15">January 15, 2024</time></p>
                <p>Article content goes here...</p>
            </article>
        </section>

        <aside>
            <h3>Related Links</h3>
            <p>Additional resources and information.</p>
        </aside>
    </main>

    <footer>
        <p>&copy; 2024 Company Name. All rights reserved.</p>
    </footer>
</body>
</html>
