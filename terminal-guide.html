Available Commands
  • pwd - show current directory
  • ls - list files and directories
  • mkdir - create directories
  • touch - create files
  • echo - display text
  • cat - read file contents
  • cd - change directory
  • help - show help
Try These Examples

  • cd test-dir
  • touch hello.txt
  • echo "Hello!" > hello.txt
