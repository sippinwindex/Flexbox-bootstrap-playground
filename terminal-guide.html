Getting Started with Terminal

Beginner

The terminal (also called command line or shell) is a text-based interface for interacting with your computer. It's incredibly powerful and essential for developers, system administrators, and power users.

Navigation Commands

Beginner

These commands help you move around the file system and understand where you are.

pwd - Show Current Directory

Print Working Directory
Terminal
student@learning:~$ pwd
/home/student
student@learning:~$ cd documents
student@learning:~/documents$ pwd
/home/student/documents
Bash 
# Print working directory
pwd

# Example output: /home/student/documents

# Use case: Find out exactly where you are
# in the file system hierarchy

ls - List Directory Contents

List Files and Directories
Terminal
student@learning:~$ ls
documents downloads pictures projects
student@learning:~$ ls -la
total 28
drwxr-xr-x 5 student student 4096 Jan 15 10:30 .
drwxr-xr-x 3 root root 4096 Jan 10 09:15 ..
-rw-r--r-- 1 student student 220 Jan 10 09:15 .bashrc
drwxr-xr-x 2 student student 4096 Jan 15 10:30 documents
drwxr-xr-x 2 student student 4096 Jan 15 08:45 downloads
ls -l

Long format with detailed information

ls -a

Show hidden files (starting with .)

ls -h

Human readable file sizes

ls -la

Long format + hidden files

ls -t

Sort by modification time

ls -R

Recursive listing (subdirectories)

Bash 
# Basic listing
ls

# List with details
ls -l

# List all files (including hidden)
ls -a

# List with details and hidden files
ls -la

# List with human readable sizes
ls -lh

# List specific directory
ls /path/to/directory

cd - Change Directory

Navigate Between Directories
Terminal
student@learning:~$ cd documents
student@learning:~/documents$ cd ..
student@learning:~$ cd /usr/bin
student@learning:/usr/bin$ cd ~
student@learning:~$ cd -
/usr/bin
student@learning:/usr/bin$
cd ~

Go to home directory

cd ..

Go up one directory level

cd -

Go to previous directory

cd .

Stay in current directory

cd ../../../

Go up multiple levels

cd /

Go to root directory

Bash 
# Change to specific directory
cd /path/to/directory

# Go to home directory
cd ~
# or just
cd

# Go up one level
cd ..

# Go to previous directory
cd -

# Go to root directory
cd /
Navigation Best Practices
  • Use pwd frequently to stay oriented
  • Use Tab completion to avoid typing full paths
  • Learn relative vs absolute paths: ../documents vs /home/user/documents
  • Use cd - to quickly switch between two directories

File Operations

Intermediate

Creating Files and Directories

touch, mkdir - Create Files and Directories
Terminal
student@learning:~$ mkdir my-project
student@learning:~$ cd my-project
student@learning:~/my-project$ touch README.md
student@learning:~/my-project$ touch index.html style.css script.js
student@learning:~/my-project$ ls -l
-rw-r--r-- 1 student student 0 Jan 15 10:45 README.md
-rw-r--r-- 1 student student 0 Jan 15 10:45 index.html
-rw-r--r-- 1 student student 0 Jan 15 10:45 script.js
-rw-r--r-- 1 student student 0 Jan 15 10:45 style.css
Bash 
# Create a directory
mkdir directory-name

# Create multiple directories
mkdir dir1 dir2 dir3

# Create nested directories
mkdir -p path/to/nested/directory

# Create an empty file
touch filename.txt

# Create multiple files
touch file1.txt file2.txt file3.txt

# Update timestamp of existing file
touch existing-file.txt

Copying and Moving Files

cp, mv - Copy and Move Files
Terminal
student@learning:~/my-project$ cp README.md README-backup.md
student@learning:~/my-project$ mv script.js app.js
student@learning:~/my-project$ mkdir backup
student@learning:~/my-project$ cp *.md backup/
student@learning:~/my-project$ ls backup/
README.md README-backup.md
Copy (cp) Options:
cp -r

Copy directories recursively

cp -i

Interactive mode (ask before overwrite)

cp -v

Verbose mode (show what's being copied)

Move (mv) Options:
mv -i

Interactive mode (ask before overwrite)

mv -v

Verbose mode (show what's being moved)

mv -n

No overwrite (don't replace existing files)

Bash 
# Copy file
cp source.txt destination.txt

# Copy file to directory
cp file.txt /path/to/directory/

# Copy directory recursively
cp -r source-dir/ destination-dir/

# Move/rename file
mv oldname.txt newname.txt

# Move file to directory
mv file.txt /path/to/directory/

# Move multiple files
mv file1.txt file2.txt file3.txt /destination/
Important

Be careful with mv and cp commands. They can overwrite existing files without warning (unless you use the -i flag).

Text Processing

Intermediate

Reading Files

cat, less, head, tail - Reading File Contents
Terminal
student@learning:~$ cat README.md
# My Project
This is a sample project for learning terminal commands.
## Features
- Modern web technologies
- Responsive design
student@learning:~$ head -3 README.md
# My Project
This is a sample project for learning terminal commands.
## Features
Command Purpose Best For Example
cat Display entire file Small files cat file.txt
less View file with pagination Large files less file.txt
head Show first 10 lines File beginning head -5 file.txt
tail Show last 10 lines Log files, recent content tail -f log.txt
Bash 
# Display entire file
cat filename.txt

# View file with pagination (press q to quit)
less filename.txt

# Show first 10 lines
head filename.txt

# Show first N lines
head -5 filename.txt

# Show last 10 lines
tail filename.txt

# Show last N lines
tail -20 filename.txt

# Follow file changes (useful for logs)
tail -f logfile.txt
